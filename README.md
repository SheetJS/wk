# wk

Preview spreadsheets in your terminal!

![example screenshot](ex.png)

## Installation

```bash
$ npm install -g wk
```

## Usage

```bash
$ wk test.xlsx
```

- `CTRL+C` quits the viewer.

- `?` displays the help message:

![help screenshot](help.png)

- `~` shows the worksheet selection screen:

![worksheet screenshot](ws.png)
